# Nisli

A small reactive Web Component framework.

Nisli started inside `backlog-mcp` as the viewer framework. It is now maintained
as a standalone package because it is used across projects.

## Package

- [`@nisli/core`](./packages/core)

## Install

```bash
npm install @nisli/core
```

## Development

```bash
pnpm install
pnpm test
pnpm typecheck
pnpm build
```
